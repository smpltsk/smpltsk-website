Designhouse ("we," "our," or "us") operates the SMPLTSK mobile application (the "Service"). This Privacy Policy informs you of our policies regarding the collection, use, and disclosure of personal data when you use our Service.

App Permissions

Our app requests the following permissions:

  • Camera - To take photos for task documentation
  • Photo Library/Storage - To save and access task images
  • Location - To geotag images and tasks for context
  • Notifications - To alert you about task updates
  • Internet - To sync data and enable collaboration
  • Network State - To optimize performance based on connectivity

You can manage these permissions in your device settings at any time.

Information We Collect

Information You Provide Directly

Account Information

  • Username or display name
  • Email address
  • Account credentials
  • Profile information from third-party authentication providers (Google, Apple, Facebook)
  • Profile picture (if provided)

Task Data

  • Task descriptions and titles
  • Task status (complete/incomplete)
  • Comments on tasks
  • Images uploaded for tasks
  • Sender and receiver relationship data

Communications

  • Customer support inquiries
  • Feedback and suggestions

Information Collected Automatically

Location Data

  • Geolocation data automatically attached to images and tasks (GPS coordinates)

Time Information

  • Timestamps for all tasks created, modified, or completed
  • Activity timestamps

Device Information

  • Device type and model
  • Operating system version
  • Unique device identifiers
  • Mobile network information
  • App instance ID (Firebase)

Usage Analytics

  • Number of tasks created, sent, and completed
  • Average tasks per user
  • Task categories or types

  • Session duration and frequency
  • App crashes and performance data
  • Firebase Analytics events and user properties
  • Firebase Performance Monitoring data
  • In-app purchase history (if applicable)
  • Ad interaction data (if ads are displayed)

Image Data

  • Photos taken or uploaded within the app
  • EXIF data from images (including location and time)
  • Image modifications or markup data

  • Enable task creation and sharing between users
  • Display tasks with their associated images, locations, and timestamps
  • Facilitate communication through comments
  • Authenticate users and manage accounts
  • Send push notifications about task updates

Communications

  • Send task-related notifications

  • Send promotional communications (with your consent)

Service Improvement

Data Sharing and Disclosure

We may share your information in the following circumstances:

With Other Users

  • Task senders can see receiver's completion status and comments
  • Task receivers can see sender's task details, images, and location data

Service Providers

Firebase (Google) for:

  • Cloud Firestore for data storage
  • Firebase Authentication for user accounts
  • Firebase Cloud Storage for image storage
  • Firebase Analytics for usage tracking
  • Firebase Crashlytics for crash reporting
  • Firebase Performance Monitoring
  • Firebase Cloud Messaging for push notifications
  • Firebase Remote Config for feature flags

Authentication providers:

  • Google Sign-In
  • Apple Sign-In
  • Facebook Login (if implemented)
  • Email/password authentication

Data Storage and Security

  • Your data is stored on secure cloud servers
  • We implement encryption for data in transit and at rest
  • Access to personal data is restricted to authorized personnel
  • We cannot guarantee absolute security but use industry-standard measures

Data Retention

  • Active account data is retained while your account is active
  • Deleted tasks and images are removed from our servers within 30 days
  • Analytics data is retained in anonymized form for up to 2 years
  • Crash logs are retained for up to 90 days
  • You may request account deletion at any time
  • Some data may be retained longer if required by law

Deletion
